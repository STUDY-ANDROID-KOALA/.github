## 🤙 RULE
```
1. 각자 `이름-성`으로 `Repository`를 판다.
2. `weekn_이름`으로 브랜치를 판다.
3. 문제를 해결하고 아래 컨벤션에 맞게 커밋한다.
4. 매주 `일요일 22시 30분` 전까지 해당 `Repository`에 아래 컨벤션에 맞게 PR을 올린다.
5. `일요일 22시 30분`에 디스코드에 모여 이번주 문제 풀이 PR에 대해 논의하는 시간을 갖는다.
```
<br>

## 📝 Convention
1. 문제 풀이 파일은 '플랫폼_레벨_문제_이름.kt' 형태의 네이밍을 사용한다.
2. PR 이름은 `[n주차] 이름` 형태의 네이밍을 사용한다.
3. 자신의 이름으로 된 개인 폴더에 주차별 폴더를 만들고 문제 풀이 파일을 올린다.
4. 커밋은 `[n주차] 플랫폼_레벨_문제_이름`의 컨벤션을 따른다.
